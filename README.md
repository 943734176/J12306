# J12306抢票助手
12306抢票程序JAVA版，自动登录-验证-查票-购票。只需简单的配置即可运行进行快捷抢票。

### 使用说明
##### 配置邮件 （若需要）
* 配置mail.setting邮件发件人信息，用于抢票成功发邮件通知抢票人
* ConfigConst.java配置类配置收件人邮箱

##### 配置抢票信息
* Main.java中，直接配置用户名密码及乘车相关信息即可
##### 开始抢票
* 启动Main函数开始抢票
